ss

ss is a wonderful tool to monitor/vew connections on and to your machine (server).

Just running ss shows all connections. You may filter it by socket type (tcp, udp, unix, raw etc) and on each connections state. To show current listening sockets use $ ss -l


If you use ss with watch you’ll get a real time updated list of connections.

For exampel, here we show only tcp-connections (the -t switch) that are currently connected (state connected);

$ watch -n 1 “ss -r -t4 state connected”
